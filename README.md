# Python Slot Machine

A simple **slot machine game** built in Python. The game allows a player to deposit money, choose how many lines to bet on, place bets, and spin the slot machine to win or lose money based on matching symbols.

## Features
- Deposit money to play.
- Choose the number of lines to bet on (1-3).
- Place bets per line.
- Randomly spin symbols on a 3x3 grid.
- Calculate winnings based on symbol values.
- Display winning lines and updated balance.
